# FCIS-SummerTraining-AndroidDevelopment
### FCIS's Second Year Summer Training Project - An Android Application for Browsing Cars

---

Car Room is an Android application that lets its users to browse different cars, read some information about them, and compare them to other cars of different types.

---

### How To Run:
- Build the project
- Download the .apk file to an emulator or an Android device
